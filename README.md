# Karga-Notify
Öğrencilerin aldıkları derslerle ilgili bilgileri izlemelerine yardımcı olur. Kullanıcılar, vize, final, bütünleme gibi sınav sonuçlarını ve derslerin durumunu bu uygulama üzerinden takip edebilirler. Ayrıca program, kullanıcılarına sınav sonuçları açıklandığında bildirim gönderir, böylece öğrenciler güncel bilgilere hızlı bir şekilde erişebilirler
